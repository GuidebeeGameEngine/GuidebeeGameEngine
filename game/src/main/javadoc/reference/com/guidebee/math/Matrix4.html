Class Overview

Encapsulates a column major 4 by 4 matrix. Like the Vector3 class it allows the chaining of methods by returning a reference to itself. For example: 

 Matrix4 mat = new Matrix4().trn(position).mul(camera.combined);

public static void mulVec (float[] mat, float[] vecs, int offset, int numVecs, int stride)

Multiplies the vectors with the given matrix. The matrix array is assumed to hold a 4x4 column major matrix as you can get from val. The vectors array is assumed to hold 3-component vectors.
