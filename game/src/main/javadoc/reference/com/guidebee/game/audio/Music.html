com.guidebee.game.audio.Music

Class Overview

A IMusic instance represents a streamed audio file. The interface supports pausing, resuming and so on. When you are done with using the IMusic instance you have to dispose it via the dispose() method.

IMusic instances are created via newMusic(FileHandle).

IMusic instances are automatically paused and resumed when an Application is paused or resumed. See ApplicationListener.

Note: any values provided will not be clamped, it is the developer's responsibility to do so
