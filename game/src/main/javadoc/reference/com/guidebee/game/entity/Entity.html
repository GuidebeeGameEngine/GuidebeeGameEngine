Class Overview

Simple containers of DataTraits that give them "data". The dataTrait's data is then processed by Directors.

public T getDataTrait (Class<T> dataTraitClass)

Retrieve a dataTrait from this Entity by class.

Note: the preferred way of retrieving DataTraits is using DataTraitMappers. This method is provided for convenience; using a DataTraitMapper provides O(1) access to dataTraits while this method provides only O(logn).

Parameters
dataTraitClass the class of the dataTrait to be retrieved.
