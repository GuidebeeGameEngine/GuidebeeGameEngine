Class Overview

Uniquely identifies a DataTrait sub-class. It assigns them an index which is used internally for fast comparison and retrieval. See Role and Entity.

DataTraitType is a package protected class.

You cannot instantiate a DataTraitType. They can only be accessed via #getIndexFor(Class). Each dataTrait class will always return the same instance of DataTraitType.
