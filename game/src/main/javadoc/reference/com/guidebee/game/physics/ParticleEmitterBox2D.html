Class Overview

ParticleEmitterBox2D use box2d rayCast:ing to achieve continuous collision detection against box2d fixtures. If particle detect collision it change it's direction before actual collision would occur. Velocity is 100% reflected. These particles does not have any other physical attributes or functionality. Particles can't collide to other particles.
