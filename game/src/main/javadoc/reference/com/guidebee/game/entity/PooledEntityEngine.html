Class Overview

Supports Entity and DataTrait pooling. This improves performance in environments where creating/deleting entities is frequent as it greatly reduces memory allocation.

  • Create entities using createEntity()
  • Create dataTraits using createDataTrait(Class)
  • DataTraits should implement the Pool.Poolable interface when in need to reset its state upon removal
